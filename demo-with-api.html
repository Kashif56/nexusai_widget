AI Chatbot Widget Demo with API Integration

This demo shows how to integrate the chatbot widget with your custom API endpoint.

Enter your API endpoint and API key below to connect the chatbot to your backend:

Integration Code

Here's the code to add the chatbot widget to your website:


<!-- Add the chatbot widget script to your HTML -->
<script src="chatbot-widget.js"></script>

<script>
    // Initialize the chatbot with your configuration
    document.addEventListener('DOMContentLoaded', function() {
        ChatbotWidget.init({
            apiEndpoint: 'https://your-chatbot-api.com/chat',
            apiKey: 'your-api-key',
            botName: 'AI Assistant',
            chatbotAccentColor: '#4a6cf7'
        });
    });
</script>

API Requirements

Your API should accept POST requests with this JSON format:


// Request format
{
    "message": "User's message here"
}

// Expected response format
{
    "response": "Bot's reply here"
}

The widget also accepts these alternative response field names: reply, message
